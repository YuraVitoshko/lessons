Задача №1

Задача №2

num--;
} Не віроно, останній результат буде 1

Задача №3

Дано:
for (let num = 0; num < 2; num++) {
console.log(`Число: ${num}`);
}
Через while:
let num = 0;
do {
console.log(`Число: ${num}`);
num++;
} while (num < 2);

ЧИСЛА (ДЗ)

Задача №1

let numOne = Math.round(1.005 * 100) / 100;
console.log(numOne);
Відповідь:
let sourceNum = 1.005 + Number.EPSILON;
let numOne = Math.round(sourceNum * 100) / 100;
console.log(numOne);

Задача №2
